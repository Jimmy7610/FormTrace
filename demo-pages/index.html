FormTrace

Demo pages for manual testing of the extension

🔒
Disabled Submit Button
The submit button stays permanently disabled. Clicking it should trigger detection of a disabled button.
Expected: Submit button was disabled
👻
Hidden Required Field
A required field is hidden via CSS. Submitting should fail silently and FormTrace should detect the hidden field.
Expected: Hidden required field blocked submission
⚠️
Invisible Error Message
Validation fails but the error message is hidden with CSS. FormTrace should detect validation without visible feedback.
Expected: Validation failed without visible feedback
🌐
Failed API Request
Submitting triggers a fetch to a non-existent endpoint. FormTrace should detect the network failure.
Expected: Network request failed after submit
✅
Successful Form
A working form that submits correctly. FormTrace should report no or low-severity issues.
Expected: No clear failure detected

How to test

1
Open a demo page in Chrome
2
Click the FormTrace extension icon to open the popup
3
Click Start recording
4
Interact with the form (fill fields, click submit)
5
Click Stop & analyze
6
Read the analysis and copy the report as Markdown
